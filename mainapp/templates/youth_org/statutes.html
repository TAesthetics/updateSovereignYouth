{% trans 'Statuten' %}

{% trans 'des Vereins „The Sovereign Youth“' %}

{% trans '§1 Name, Sitz und Tätigkeitsbereich' %}

  1. {% trans 'Der Verein führt den Namen „The Sovereign Youth“.' %}
  2. {% trans 'Er hat seinen Sitz in Salzburg und erstreckt seine Tätigkeit auf ganz Österreich und Europa.' %}
  3. {% trans 'Die Errichtung von Zweigvereinen und lokalen Gruppen ist vorgesehen.' %}

{% trans '§2 Zweck des Vereins' %}

  1. {% trans 'Der Verein ist nicht auf Gewinn gerichtet und verfolgt ausschließlich gemeinnützige Zwecke.' %}
  2. {% trans 'Ziel ist die Förderung einer wertebasierten Jugendkultur auf Grundlage von geistiger Freiheit, Eigenverantwortung und europäischer Kultur.' %}
  3. {% trans 'Der Verein ist überparteilich und unabhängig.' %}
  4. {% trans 'Die Tätigkeit gliedert sich in zwei klar getrennte Bereiche:' %}
    • {% trans 'Sovereign Students – schulpolitisch und gesellschaftlich aktiv' %}
    • {% trans 'The Digital Coil – bildungs- und technikorientiertes Lernprojekt' %}

{% trans '§3 Mittel zur Erreichung des Vereinszwecks' %}

{% trans 'Die Ziele werden verwirklicht durch:' %}

  • {% trans 'Veranstaltungen, Workshops, Seminare und Diskussionen' %}
  • {% trans 'Online-Formate, Lernzirkel, Publikationen' %}
  • {% trans 'Öffentlichkeitsarbeit und Kampagnen' %}
  • {% trans 'Aufbau eines Netzwerks junger engagierter Menschen' %}
  • {% trans 'Zusammenarbeit mit Schulen, Organisationen und freien Initiativen' %}

{% trans '§4 Mitgliedschaft' %}

  1. {% trans 'Mitglied kann jede natürliche Person werden, die die Ziele unterstützt.' %}
  2. {% trans 'Mitgliedsarten:' %}
    • {% trans 'Aktivmitglieder (14–35 Jahre)' %}
    • {% trans 'Fördermitglieder (ab 36 Jahren oder unterstützend)' %}
    • {% trans 'Ehrenmitglieder (auf Vorschlag durch die Generalversammlung)' %}
  3. {% trans 'Voraussetzung: Zustimmung zu den Grundwerten (Demokratie, Menschenwürde, geistige Freiheit, Eigenverantwortung).' %}

{% trans '§5 Rechte und Pflichten der Mitglieder' %}

  1. {% trans 'Teilnahme an Vereinsaktivitäten.' %}
  2. {% trans 'Aktivmitglieder besitzen Stimmrecht in der Generalversammlung.' %}
  3. {% trans 'Mitglieder verpflichten sich zur Einhaltung der Statuten und Unterstützung des Vereinszwecks.' %}

{% trans '§6 Organe des Vereins' %}

  • {% trans 'Generalversammlung' %}
  • {% trans 'Vorstand' %}
  • {% trans 'Rechnungsprüfer:innen' %}
  • {% trans 'Bereichsleitungen (optional)' %}

{% trans '§7 Generalversammlung' %}

  1. {% trans 'Höchstes Organ, mind. einmal jährlich.' %}
  2. {% trans 'Zuständig für:' %}
    • {% trans 'Wahl/Entlastung des Vorstands' %}
    • {% trans 'Statutenänderungen' %}
    • {% trans 'Grundsatzentscheidungen' %}
    • {% trans 'Auflösung' %}

{% trans '§8 Vorstand' %}

  1. {% trans 'Besteht aus mindestens drei Personen:' %}
    • {% trans 'Obmann/Obfrau' %}
    • {% trans 'Kassier:in' %}
    • {% trans 'Schriftführer:in' %}
    • {% trans '(Optional: Stellvertretung für Obmann/Obfrau)' %}
  2. {% trans 'Führt laufende Geschäfte, vertritt den Verein nach außen.' %}
  3. {% trans 'Funktionsperiode: zwei Jahre, Wiederwahl möglich.' %}

{% trans '§9 Rechnungsprüfer:innen' %}

  1. {% trans 'Zwei Personen, gewählt von der Generalversammlung für zwei Jahre.' %}
  2. {% trans 'Prüfen Finanzen, berichten der Generalversammlung.' %}

{% trans '§10 Auflösung des Vereins' %}

  1. {% trans 'Nur mit Zweidrittelmehrheit in einer eigens einberufenen Generalversammlung.' %}
  2. {% trans 'Vereinsvermögen fällt an eine gemeinnützige Organisation mit ähnlichem Zweck.' %}
